# Images

Dashboard deliverables live here. The `.pbix` is built in Power BI Desktop from [`../BUILD_GUIDE.md`](../BUILD_GUIDE.md); once it's done this folder holds:

- **Screenshot(s)** of the finished single-page dashboard (PNG, e.g. `dashboard.png`), embedded in the project [`../README.md`](../README.md).
- The **`.pbix` file** itself, so the project is fully reproducible and recruiters can open it.

Tip: a clean full-page screenshot (Windows: `Win + Shift + S`) at a readable resolution works best for the README preview.
