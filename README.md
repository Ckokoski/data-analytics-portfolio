## How this portfolio was built

This portfolio was built using an **AI-assisted development workflow.** I directed the architecture and own every analytical decision, interpretation, and finding; AI handled scaffolding, boilerplate, and first-draft documentation. Directing AI to do real, accountable work is the skill set I spent five years building in marketing technology — so I'm showing it openly rather than hiding it.

Every dataset here is either **public** (with its source and license noted) or **clearly-labeled synthetic** — except my **real** YouTube channel data in Project 02. Nothing in this repo claims work I didn't do, and self-directed projects are labeled as such.
